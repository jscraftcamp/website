How to Register for JSCraftCamp

JSCraftCamp uses a unique community-driven registration process. Instead of a traditional form, you'll create a simple JSON file about yourself and submit it through GitHub. This helps us build our participant page and gives you a taste of collaborative workflows used in the JavaScript community.

1

Create a JSON File

Create a JSON file with your information following our template format. The file should be named firstname_lastname.json.

You can click the "Register Now" button above to get started with a pre-filled template on GitHub.

2

Submit a Pull Request

Submit your file as a pull request to our GitHub repository. This adds your registration to our system.

Don't worry if you're new to GitHub - we're happy to help you through the process!

3

Confirmation

Once your pull request is accepted, you're officially registered! Your name will appear on our participants page.

You'll receive updates about the event through GitHub and our community channels.

Need Help?

If you're not familiar with GitHub or pull requests, don't worry! Reach out on Discord, there is a community to guide you through the process.

Registration JSON Format

Your registration JSON file should include the following information. The comments explain each field:

// vim: ft=jsonc // Please provide your info in your own .json file. // See https://jscraftcamp.org/registration for more information // This is the template for your registration { // Your real name (required by venue) "realName": { "givenName": "Your Given Name", "familyName": "Your Family Name", // if you prefer to have your family name shown first (optional) "placeFamilyNameFirst": false, // if you do not want to show your family name on the website (optional) "hideFamilyNameOnWebsite": false }, // Your GitHub username "githubAccountName": "yourgithubhandle", // Your company name (optional) "company": "Your Company", // Which days you'll attend (at least one must be true) "when": { "friday": true, // June 27th, 2025 "saturday": true // June 28th, 2025 }, // Are you willing to take session notes? (required) "iCanTakeNotesDuringSessions": true, // Your JavaScript interests/skills (required) "tags": ["JavaScript", "React", "Node.js", "Testing"], // Dietary preferences (optional) "vegan": false, "vegetarian": false, "allergies": ["List allergies here"], // Tell us about your JavaScript journey (required) "whatIsMyConnectionToJavascript": "...", // What you can bring to the community (required) "whatCanIContribute": "...", // T-shirt size if you want one (optional) "tShirtSize": "M" }

Important Registration Notes

Field Notes
Real Name Your real name is required for venue security. ID may be checked at entrance.
Attendance Days You must select at least one day (Friday/Saturday) to attend.
Taking Notes As a community event, we ask all participants to be willing to take notes during at least one session.
Connection to JavaScript Tell us a bit about your experience with JavaScript and what you're hoping to learn/share.
Dietary Needs We'll do our best to accommodate dietary restrictions, but cannot guarantee all needs will be met. If you have severe allergies, please consider bringing backup food options.
Social Links Optional links to your social profiles will be displayed on the participants page.

Need to Cancel Your Registration?

We understand plans change. If you can no longer attend JSCraftCamp, please let us know as soon as possible so we can offer your spot to someone else. You can cancel your registration in any of these ways:

📝

Submit a PR

Create a pull request to remove your JSON file from the repository

🔔

Open an Issue

Create an issue in our GitHub repository asking to be removed

✉️

Email Us

Send a message to team@jscraftcamp.org

Registration is Free

JSCraftCamp is completely free to attend thanks to our sponsors. However, since spots are limited, please register only if you're confident you can attend. If your plans change, let us know promptly so others can participate.
